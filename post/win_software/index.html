Windows软件推荐

发表于 2019-05-16 | 更新于 2019-05-20 | 分类于 share

整理一些Windows平台好用的软件，目前包括：Bitdefender，Windows Defender，PotPlayer，Typora，Everything，Recuva，CCleaner，Chrome，Bandizip，OneQuick，GoodSync，TeamViewer

Bitdefender

免费版：

https://www.bitdefender.com/solutions/free.html

Bitdefender家的付费杀软是地表最强的，免费版同样值得信赖。轻巧，静默，不刷存在感，缺点是可定制功能少，没有附带工具箱，英文界面，经常误杀破解软件和小厂软件，需要手动加到信任路径。

Windows Defender

win10自带，日常足够。缺点是经常误杀破解软件，会定时扫描硬盘（可以在组策略里关闭），以及对系统性能和续航有一点影响。

PotPlayer

视频播放器，功能强大

http://potplayer.daum.net/?lang=zh_CN

Typora

win平台最棒的Markdown写作软件

https://www.typora.io/

Everything

快速查找电脑上的文件

https://www.voidtools.com/zh-cn/

Recuva

找回误删的文件

https://www.ccleaner.com/recuva/

CCleaner

清理垃圾文件和隐私

https://www.ccleaner.com/ccleaner/

Chrome

浏览器首推

https://www.google.cn/intl/zh-CN/chrome/

Bandizip

压缩软件

http://www.bandisoft.com/bandizip/

或：360压缩 http://yasuo.360.cn/

OneQuick

强大方便的热键快捷工具

https://onequick.org/

GoodSync

文件同步和备份软件

https://www.goodsync.com/cn

TeamViewer

远程连接和远程控制软件

https://www.teamviewer.com/cn/
